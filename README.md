# Python+OpenCV 人脸识别系统 python866

该项目致力于通过技术手段实现人脸识别的数字化与智能化，提高工作效率，并满足不同用户角色对系统的需求。

## 一、项目概述

### 核心目标
旨在将人脸识别的数据管理从手工记录转变为网络化信息管理，以适应现代化工作的需求。

### 用户角色
- 管理员：具备用户信息、公告信息、操作日志的管理权限。
- 普通用户：拥有账号注册登录、查看公告、修改个人信息的权限。

## 二、技术选型与环境

### 技术栈
- **开发语言**：Python，以其简单易学、高效执行和面向对象编程的特点。
- **框架**：Django，采用 MVT 架构，简化开发流程，提高开发效率。
- **架构**：B/S（浏览器/服务器），便于用户通过浏览器访问，无需安装额外客户端。
- **数据库**：MySQL，以其低成本、小巧、快速的特点，满足中小型系统的数据存储需求。
- **核心功能技术**：OpenCV，提供人脸识别技术支持。

### 运行与开发环境
- **开发工具**：PYCHARM。
- **运行环境**：Jre、Tomcat；操作系统兼容 Windows 2000/2003/NT/XP/Vista/Windows 7 及以上中文系统。
- **硬件要求**：CPU 双核以上 P42.0G 及以上、内存 256M 及以上、硬盘 40G 及以上、10M/100M / 千兆网卡。

## 三、功能模块

- 用户管理：支持用户信息的新增、删除、搜索。
- 公告管理：支持公告的发布和修改。
- 操作日志：查看和管理用户的操作记录。
- 个人信息管理：支持普通用户修改个人信息。

## 五、核心亮点

- **智能化**：采用先进的人脸识别技术，提升管理效率。
- **易用性**：基于B/S架构设计，用户无需安装额外软件即可使用。
- **扩展性**：基于Python和Django开发，便于后续的功能扩展与维护。
